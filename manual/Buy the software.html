Buy the software

If you touch "Proceed to purchase screen" on the software information screen 6 or the search result screen 5, you will proceed to the purchase screen. Follow the on-screen instructions to purchase.

  • * After purchasing, the software will start downloading automatically. You can check the download status of the software by clicking "Download Management" in the HOME button menu.
  • * The purchased software can be used not only by the user who purchased it, but also by all users registered on the same console.
