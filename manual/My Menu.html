The software registered in the "Interest List" will be displayed.

When you touch the "View List" button, the software registered in the "Interest List" is displayed in a list.

Recommendation Polls

The software that you can vote for will be displayed.

If you touch the software icon and answer three questions such as recommendation, you can recommend the software you played to everyone.

When you touch the "View list" button, a list of software that can be voted for is displayed.

The results of the tabulation of the voting contents are displayed in software information 6 and ranking 12.

* You can only vote once per user per software. You can vote on the software you played for a total of more than 1 hour.

* If the user's age is not included in the software's age category, you will not be able to vote.

* Some software cannot be voted on.

* You can check the recommended contents in the "Recommended Voting History".
